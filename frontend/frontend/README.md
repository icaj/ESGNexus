# Frontend Streamlit

Frontend profissional do ESG Nexus Enterprise.

## Execução

```bash
streamlit run frontend_streamlit/app.py
```

Acesse:

```text
http://localhost:8501
```

## Funcionalidades

- Login e cadastro de usuário via API.
- Modo demonstração sem API.
- Dashboard Executivo ESG.
- Dashboard de Machine Learning.
- Cadastro de fornecedores.
- Classificação ESG.
- Explicabilidade de IA.
- Tela operacional para Airflow e MLflow.

## Configuração

Variável para apontar a API:

```env
ESG_NEXUS_API_URL=http://localhost:8000
```

## Design

As telas foram desenhadas sem emojis, com visual corporativo e foco em análise gerencial.
