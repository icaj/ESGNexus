# ESG Nexus Frontend

Frontend da solução ESG Nexus em Next.js.

## Requisitos
- Node.js 20+
- npm

## Configuração
Crie o arquivo `.env.local` com base em `.env.example`.

Exemplo:
```env
NEXT_PUBLIC_API_URL=http://localhost:8080/api
```

## Executar localmente
```bash
npm install
npm run dev
```

## Telas entregues
- Login
- Dashboard
- Fornecedores
- Ranking
- Alertaas
- Configurações
